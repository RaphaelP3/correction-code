# Projet scolaire - Qualité de code Front-End

## Objectif 
Analyser, tester et corriger la qualité du code HTML, CSS et JavaScript d'une copie d'un site.

## Technologies 
- HTML 
- CSS
- JavaScript
- Git / GitHub


## Outils de qualité de code
Pour assurer le code propre et conforme aux standards, les outils suivants peuvent être utilisés : 
- **HTMLHint** - Vérification des erreurs et bonnes pratiques HTML
- **Stylelint** - Analyse et correction du CSS
- **ESlint** - Détection d'erreurs et amélioration du Javascript
